# Dev

1. Clone project
2. Copy  ```env.template``` y rename to ```.env```
3. Run
```
npm install
```
4. create container (Docker desktop)
```
docker-compose up -d
```

5. run backend
```
npm run start:dev
```

6. go to site
```
localhost:3000/graphql
```
